# RADIX Robotics — Composite Labs for Schools

RADIX Robotics provides state-of-the-art composite laboratories for educational institutions. Our labs are designed to inspire the next generation of engineers and scientists through hands-on learning with advanced robotics kits, 3D printing technology, and immersive simulations.

## Features
- **3D Interactive Visualizations**: Powered by Three.js for realistic robot models.
- **Dynamic Animations**: Smooth scroll-triggered animations with GSAP.
- **Modern Design**: A high-performance, responsive UI built for modern education.

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Libraries**: [GSAP](https://gsap.com/), [Three.js](https://threejs.org/), [ScrollTrigger](https://gsap.com/docs/v3/Plugins/ScrollTrigger/)
- **Fonts**: Syne, Orbitron, DM Sans
